== Description ==
The Simple markers map Plugin Gutenberg Block is a WordPress plugin that enables you to easily add customizable maps to your posts and pages using the Gutenberg editor. This plugin is develop with <a href="https://leafletjs.com/" target="_blank">LeafletJS</a> an open-source JavaScript library, <a href="https://www.openstreetmap.org/" target="_blank">OpenStreetMap®</a> a open data, <a href="https://www.google.com/maps/" target="_blank">Google Maps Layers</a>.

== Features ==
Easy Integration: Add Maps directly from the Gutenberg editor without API key.
Customizable Settings: Adjust map settings such as zoom level, map type, and marker positions.
Multiple Markers: Add multiple markers to highlight various locations on a single map.
User-Friendly Interface: Intuitive controls and settings for effortless map customization.

== Installation ==
Go to your WordPress dashboard. Install the plugin either via the WordPress plugin directory or by navigating to Plugins > Add New > Upload Plugin. Choose the downloaded zip file and click Install Now. Then, activate the plugin.

== Frequently Asked Questions ==
How to add a map to a page?
In the Gutenberg editor, click the + button to add a new block.
Search for "Simple markers map" and add the block to your post or page.
